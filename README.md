# Atm-Banking
This is a simple console-based ATM system developed using Java. It simulates basic banking operations in ATM 

 Features
* Login using last 4 digits of account number and PIN
* Deposit money
* Withdraw money
* Transfer money between accounts
* Fast cash withdrawal options
* View mini statement with transaction history
* Check account balance
---------

 Tools Used

Java
ArrayList
Scanner
LocalDateTime
------------------
 How to Run
Compile the program
Run: java BankSystem
